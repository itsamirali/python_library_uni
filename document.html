Capabilities

The Requests library is packed with powerful features that make HTTP requests simple and intuitive:

Core Features

  • International Domains and URLs
  • Keep-Alive & Connection Pooling
  • Sessions with Cookie Persistence
  • Browser-style SSL Verification
  • Basic/Digest Authentication
  • Elegant Key/Value Cookies
  • Automatic Decompression
  • Automatic Content Decoding
  • Unicode Response Bodies
  • HTTP(S) Proxy Support
  • Multipart File Uploads
  • Streaming Downloads
  • Connection Timeouts
  • Chunked Requests
  • `.netrc` Support

HTTP Features Support

Feature Description
Authentication Basic, Digest, Custom, OAuth
Compression gzip, deflate, brotli
Proxies HTTP(S), SOCKS
Custom Headers Full support for custom HTTP headers
Persistent Connections Connection pooling and reuse
Sessions Cookie persistence, connection pooling

Data Handling

# Automatic JSON handling
response = requests.get('https://api.example.com/data')
json_data = response.json()

# Form data
requests.post('https://api.example.com/submit', data={'key': 'value'})

# File uploads
files = {'file': open('report.pdf', 'rb')}
requests.post('https://api.example.com/upload', files=files)

# Streaming large files
with requests.get('https://example.com/large-file', stream=True) as r:
    with open('large-file.zip', 'wb') as f:
        for chunk in r.iter_content(chunk_size=8192):
            f.write(chunk)

Installation

Installing Requests is simple with pip:

$ pip install requests

Requests officially supports Python 2.7 & 3.6+

Quickstart

Make a simple request:

import requests

# Make a GET request
r = requests.get('https://api.github.com/user')
print(r.status_code)  # 200
print(r.json())

HTTP Methods

GET Request

requests.get('https://api.github.com/user', 
    params={'key': 'value'},
    headers={'Authorization': 'token xxx'})

POST Request

requests.post('https://api.example.com/post',
    json={'key': 'value'},
    headers={'Content-Type': 'application/json'})

Other Methods

requests.put('https://api.example.com/put')
requests.delete('https://api.example.com/delete')
requests.head('https://api.example.com/head')
requests.options('https://api.example.com/options')

Response Objects

Attribute Description
r.text Response content as unicode string
r.content Response content as bytes
r.json() Response parsed as JSON
r.status_code HTTP status code
r.headers Response headers

Advanced Usage

Session Objects

with requests.Session() as s:
    s.auth = ('user', 'pass')
    s.headers.update({'x-test': 'true'})
    
    # Both requests will use the same session
    r = s.get('https://api.example.com/get')
    r = s.post('https://api.example.com/post')

Request Timeouts

# Timeout in seconds
requests.get('https://api.github.com', timeout=3)

Error Handling

try:
    r = requests.get('https://api.example.com')
    r.raise_for_status()
except requests.exceptions.RequestException as e:
    print(f"An error occurred: {e}")
